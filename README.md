# SambaRunner
A simple GUI to run or stop smbd oder nmbd. 
It allows to check the status of smbd and nmbd.
<br> 

#### Requierment: <br />
```
Qt 5.14 (or higher)
```
Download Qt here: https://www.qt.io/download
